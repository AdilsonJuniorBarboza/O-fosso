REGRAS

  • Para participar do ranqueamento é necessário nos mandar um vídeo realizando qualquer fosso tier 80+ .
  • É um ranqueamento brasileiro , logo, aqui é das ruas, pode usar elixir, pode usar bug, pode usar tudo, contanto que o vídeo fazendo o fosso seja enviado no nosso Discord.
  • Para participar basta acessar o nosso Discord e enviar o link do seu vídeo (que deve ser postado no YouTube) preenchendo o formulário corretamente.
  • Caso você for enviar outro vídeo fazendo um tier maior com a mesma classe, deixar enaltecido no formulário que se trata de uma ATUALIZAÇÃO.
  • Você pode encaminhar seu vídeo até a data do final da temporada, ou seja, você tem até o dia 31 de Julho para participar da ranqueada.
  • Iremos realizar uma premiação para os primeiros colocados de suas respectivas classes e para os 3 primeiros do rank geral. Porém, estamos buscando parceiros para tornar essa "competição" mais atrativa para os participantes.
